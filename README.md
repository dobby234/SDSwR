`floodUncertR` is a lightweight R package providing a reproducible
workflow for terrain-based flood susceptibility analysis. It derives
terrain characteristics from a Digital Elevation Model (DEM), computes a
simple terrain-based flood susceptibility index, and visualizes the
result together with its uncertainty.

Analysis state is tracked through a `flood_project` S3 object that is
created, updated, and inspected as it moves through the workflow.

The package is intended as a lightweight teaching and exploratory-analysis
toolkit, not as operational flood forecasting software.

## Installation

You can install the development version of floodUncertR from
[GitHub](https://github.com/dobby234/floodUncertR) with:

``` r
# install.packages("remotes")
remotes::install_github("dobby234/floodUncertR", build_vignettes = TRUE)
```

## Quick example

``` r
library(floodUncertR)

# Synthetic DEM, so the example runs with no external data
dem <- terra::rast(matrix(runif(900, 0, 200), nrow = 30))
project <- new_flood_project(dem = dem)

print(project)
summary(project)

project <- derive_terrain_metrics(project)
project <- compute_flood_index(project)
project <- plot_uncertainty(project)
```

See `vignette("floodUncertR")` for the full walkthrough with explanations
of each step.

## Workflow

1. Define a study area (optional, `sf` polygon or bounding box).
2. Retrieve a DEM with `get_dem()` (requires the `elevatr` package and
   an internet connection), or supply your own `SpatRaster`.
3. Create a `flood_project` object with `new_flood_project()`.
4. Derive terrain metrics with `derive_terrain_metrics()`.
5. Compute a flood susceptibility index with `compute_flood_index()`.
6. Visualize susceptibility and uncertainty with `plot_uncertainty()`.

## Function reference

| Function                    | Description                                                             |
|------------------------------|--------------------------------------------------------------------------|
| `new_flood_project()`         | Construct a `flood_project` S3 object from a DEM (and optional study area) |
| `print.flood_project()`       | Print a compact overview of which workflow steps are complete           |
| `summary.flood_project()`     | Print DEM stats, elevation range, and workflow status                   |
| `is_flood_project()`          | Check whether an object is a `flood_project`                            |
| `get_dem()`                   | Download a DEM for a study area via `elevatr` (requires internet)       |
| `derive_terrain_metrics()`    | Compute slope, aspect, and flow direction from the DEM                  |
| `compute_flood_index()`       | Compute a terrain-based flood susceptibility index                      |
| `plot_uncertainty()`          | Plot the susceptibility index alongside a local uncertainty layer       |
